문제

한 글자를 입력받아 그 값이 알파벳 소문자, 알파벳 대문자, 숫자, 기호 중 무엇인지를 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
한 글자를 입력하세요: 2
숫자를 입력했군요^^ 


한 글자를 입력하세요: A
알파벳 대문자를 입력했군요^^ 


한 글자를 입력하세요: !
기호를 입력했군요^^

코드 빈칸 채우기

#include <stdio.h> 
int main() { 
	printf( "한 글자를 입력하세요: " ); 
	char letter = '\0'; 
	scanf( "%c", &letter ); 
 
	 (  ( 'a'  letter )  ( letter  'z' ) ) { 
		printf( "알파벳 소문자를 입력했군요^^" ); 
	} 
	 (  ( 'A'  letter )  ( letter  'Z' ) ) { 
		printf( "알파벳 대문자를 입력했군요^^" ); 
	} 
	 (  ( '0'  letter )  ( letter  '9' ) ) { 
		printf( "숫자를 입력했군요^^" ); 
	} 
	 { 
		printf( "기호를 입력했군요^^" ); 
	} 
	 
	return 0; 
}
