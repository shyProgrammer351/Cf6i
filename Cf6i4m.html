문제

동전을 임의로 던지면 어느 면이 위쪽을 향하도록 떨어졌는지를 사용자가 맞추는 동전 던지기 게임을 여러 번 계속할지 묻는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
동전의 앞면(1), 뒤면(2) 중 하나를 선택하세요: 1
맞췄습니다.
동전 던지기 게임을 한 번 더 하시겠습니까?(y/n): y
동전의 앞면(1), 뒤면(2) 중 하나를 선택하세요: 1
틀렸습니다.
동전 던지기 게임을 한 번 더 하시겠습니까?(y/n):n

프로그램 코드

	#include <stdio.h>
	#include <stdlib.h>							
	#include <time.h>							
	
1	int main() {
		int headsOrTails = 0, user = 0;
2		char again = '\0';
	
3		do {
4			srand( time( NULL ) );						
5			headsOrTails = rand() % 2 + 1;					
	
6			printf("동전의 앞면(1), 뒤면(2) 중 하나를 선택하세요: ");
7			scanf( "%d", &user );
	
8			( headsOrTails == user ) ? printf( "맞췄습니다.\n" ) : printf("틀렸습니다.\n");
	
9			printf("동전 던지기 게임을 한 번 더 하시겠습니까?(y/n): ");
10			scanf( "\n%c", &again );
11		} while( ( again == 'y' ) || ( again == 'Y' ) );
	
12		return 0;
	}
