문제

A양과 B군이 가위바위보를 선택하면 승패를 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
A양이 이겼습니다. 


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 3
보를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 1
가위를 냈습니다.
B군이 이겼습니다. 


A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: 2
바위를 냈습니다.
A양과 B군이 비겼습니다.

코드 빈칸 채우기

#include <stdio.h> 
int main() { 
	printf( "A양, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 
	 playerA = 0; 
	scanf( "%d", &playerA ); 
	( playerA ) { 
	 1 										 
		printf( "가위를 냈습니다.\n" );  
		;									 
	 2 										 
		printf( "바위를 냈습니다.\n" );  
		;									 
	 3 										 
		printf( "보를 냈습니다.\n" );  
	} 
 
	printf( "B군, 가위(1), 바위(2), 보(3) 중 하나를 숫자로 입력하세요: " ); 
	 playerB = 0; 
	scanf( "%d", &playerB ); 
	( playerB ) { 
	 1 										 
		printf( "가위를 냈습니다.\n" ); 	 
		;									 
	 2 										 
		printf( "바위를 냈습니다.\n" );  
		;									 
	 3 										 
		printf( "보를 냈습니다.\n" );  
	} 
 
	 ( playerA  playerB ) { 
		printf( "\nA양과 B군이 비겼습니다.\n" );  
	} 
	 ( ( playerA  1 )  ( playerB  3 ) ) {	 
		printf( "\nA양이 이겼습니다.\n" );  
	} 
	 ( ( playerA  2 )  ( playerB  1 ) ) { 
		printf( "\nA양이 이겼습니다.\n" );  
	} 
	 ( ( playerA  3 )  ( playerB  2 ) ) { 
		printf( "\nA양이 이겼습니다.\n" );  
	} 
	 { 
		printf( "\nB군이 이겼습니다.\n" );  
	} 
 
	return 0; 
}
